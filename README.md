# C# Projects

These are projects I created while learning C# and the .NET Framework.

## Projects

### Car Insurance MVC Application

This is an ASP.NET MVC web application that simulates a car insurance website. Users enter policy information and the application calculates an insurance quote based on business logic. There is also an admin page where staff can view all issued quotes.

### TwentyOne Game

This project is a C# console application that simulates the Blackjack card game. It demonstrates object-oriented programming concepts such as classes, inheritance, and lists.
